### Task-Tracker

# I built a Task Tracker application after learning React.js to gain hands-on experience with core concepts.

Package Installation:

- mongoose
- cors
- express
- nodemon (optional)

Funtionality 
- create an task
- fetch all the tasks from database
- checkbox(done/undone)
- delete a completed task


Tech Stack:
- Frontend - react.js
- Backend - Node js and express js
- Database - MongoDB
